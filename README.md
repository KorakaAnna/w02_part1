# w02_part1



1️. Κατανόηση της Αρχιτεκτονικής
Η εφαρμογή μας ακολουθεί την τριών επιπέδων (three-tier) αρχιτεκτονική:
1.	Frontend (React Native): UI και αλληλεπίδραση με τον χρήστη.
2.	Backend (Node.js + Express): Διαχείριση αιτημάτων και επιχειρησιακή λογική.
3.	Database (MariaDB): Αποθήκευση και ανάκτηση δεδομένων.
Άσκηση 1: Σχεδίαση Αρχιτεκτονικής
1.	Χρησιμοποιήστε χαρτί ή ένα ψηφιακό εργαλείο (π.χ. draw.io) για να σχεδιάσετε τη ροή της εφαρμογής σας.
2.	Εντοπίστε τα βασικά endpoints που θα χρειαστείτε:
o	/users/register (Εγγραφή χρήστη)
o	/users/login (Σύνδεση χρήστη)
o	/restaurants (Λίστα εστιατορίων)
o	/reservations (Δημιουργία κράτησης)
3.	Αποθηκεύστε το διάγραμμα και ανεβάστε το στο GitHub.
